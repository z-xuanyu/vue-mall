# vue + vant ui 实战mall移动商城



### 项目启动安装所有依赖包
```
npm install
```

### 项目运行
```
npm run serve
```

### 项目打包上线
```
npm run build
```

### 项目运行在测试环境
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
